# AI中医营养师 推广方案

## 一、产品定位

**产品名称**：AI中医营养师
**产品类型**：微信小程序 / Web应用
**核心功能**：中医体质测试 + 个性化食材推荐 + 时令养生指导
**目标用户**：25-45岁城市白领、亚健康人群、养生爱好者

---

## 二、用户画像与痛点分析

### 2.1 主要用户群体

| 用户群体 | 特征 | 需求 |
|---------|------|------|
| 城市白领 | 工作压力大、饮食不规律 | 快速了解自身体质，获取简单易行的养生建议 |
| 亚健康人群 | 湿气重、脾胃虚、易疲劳 | 希望通过食疗辅助调理 |
| 养生爱好者 | 相信中医理念、主动学习 | 寻找靠谱的中医科普和个性化推荐 |
| 健身/减脂人群 | 想要科学饮食指导 | 结合体质制定饮食方案 |

### 2.2 核心痛点

1. **不知道自己是什么体质** - 盲目养生，效果不佳
2. **不知道该吃什么不该吃什么** - 网络信息杂乱，真假难辨
3. **缺乏便捷的查询工具** - 买菜、做饭时无法快速查询
4. **不知道季节变化该如何调理** - 养生知识不成体系
5. **缺乏持续使用的动力** - 应用打开一次就卸载

## 三、核心推广方案

### 3.1 微信生态推广

#### 3.1.1 微信小程序优势
- 无需下载、即用即走
- 微信生态闭环，裂变传播方便
- 背靠12亿月活用户

#### 3.1.2 推广方式

```
【入口一】公众号关联
- 现有公众号菜单接入小程序
- 图文消息内嵌小程序卡片
- 关注自动回复小程序链接

【入口二】朋友圈分享
- 用户分享体质报告到朋友圈
- 报告设计精美，有分享动力
- 附带小程序码，引导更多人使用

【入口三】微信群推广
- 组建养生打卡群
- 群内分享养生知识
- 邀请注册双方获得积分奖励

【入口四】附近的小程序
- 开启地理位置展示
- 覆盖周边5公里用户
- 关键词优化提高曝光
```

### 3.3 内容营销策略

#### 3.3.1 平台选择

| 平台 | 内容形式 | 目标 |
|------|---------|------|
| 微信公众号 | 深度图文 + 测试类文章 | 沉淀用户 |
| 知乎 | 专业科普 + 问答 | 建立权威 |
| 小红书 | 养生Plog + 食谱分享 | 年轻用户 |
| 抖音/视频号 | 短视频 + 直播 | 广泛曝光 |
| B站 | 知识区视频 | 年轻高学历用户 |

#### 3.3.2 爆款内容方向

**标题方向**：
- 《9道题测出你的中医体质，我是气虚质，你呢？》
- 《湿气重的人注意！这些食材要少吃》
- 《春季养生必看！这5种食材要多吃》
- 《气虚体质怎么调？老中医推荐3道食疗方》
- 《24节气养生大全，建议收藏》

**内容形式**：
- 体质测试类 → 引发好奇，引导测试
- 食材科普类 → 实用价值，收藏分享
- 节气养生类 → 时效性强，季节热点
- 食谱推荐类 → 具体可操作

#### 3.3.3 KOL合作

```
合作类型：
1. 中医养生类博主（@针灸匠人 @中医老徐）
2. 美食博主（推广药食同源食谱）
3. 健身博主（结合体质推荐饮食）
4. 职场博主（针对亚健康话题）

合作形式：
- 博主测评 + 体验报告
- 视频植入小程序
- 直播连麦互动
- 专属优惠码推广
```
